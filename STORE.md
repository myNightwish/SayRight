# Nuance 语差 · 上架清单（Chrome 网上应用店）

## 一、打包
```bash
bash package.sh   # 生成 nuance-语差-v0.1.0.zip
```
脚本只收录前端文件，**已排除 `tutor-server.mjs`、`.env`** 等敏感/服务端文件。

## 二、发布前必做
1. **口令改成一人一个**：服务端已支持多口令 + 每日配额。在 Render 环境变量里设
   `TUTOR_ACCESS_KEYS=alice-x9,bob-k3,...`（逗号分隔，每个用户发不同的），
   `TUTOR_DAILY_QUOTA=200`（每口令每日上限）。谁把口令转给别人，配额会被一起刷爆、
   从日志能看出异常，直接从变量里删掉那个口令即可单独吊销。
2. **隐私政策要有公网 URL**：`privacy.html` 已随服务托管，地址为
   `https://say-50s9.onrender.com/privacy.html`，填进上架表单的隐私政策栏。
3. （可选）manifest 里的 `http://127.0.0.1:8770/*` 是本地调试用，上架版可留（无害），
   介意的话打包前删掉这一条 host_permission。

## 三、商店信息文案

**名称**：Nuance 语差

**简介（≤132 字符）**：
> 翻译加地道润色：说一句英文，AI 告诉你哪里不地道、更自然的说法是什么。收藏你的「语差」，场景预演、随机出题、主动回忆。

**详细描述**：
> Nuance 语差 帮你把「能看懂但说不地道」的英语练成脱口而出。
>
> ● 说的对吗：输入一句英文，AI 指出不自然之处，给出更地道的表达和原因，可选场景（面试/邮件/客户…）、语气、严格度。
> ● 场景预演：说出你接下来要面对的场景，先把整段对话走一遍，带走关键表达。
> ● 对话演练：跟 AI 角色实时对练，结束后复盘你可以说得更地道的地方。
> ● 收藏 + 练习：收藏地道说法与句式骨架，随机出题、复述与泛化两种模式主动回忆。
>
> 隐私：收藏与设置只存本机，没有账号。你主动点击时才会把当次输入的文本发送到服务端生成结果，服务端不落库、不保存。

**分类**：教育
**语言**：中文（简体）

## 四、隐私 / 权限（审核必填）

**隐私政策 URL**：`https://say-50s9.onrender.com/privacy.html`

**单一用途说明**：
> 帮助用户练习地道英文表达：润色点评、场景预演、对话演练与复习。

**权限理由**：
| 权限 | 理由文案 |
|---|---|
| `storage` | 在本机保存用户的收藏与偏好设置，不上传。 |
| `clipboardWrite` | 让用户一键复制润色结果。 |
| `host: say-50s9.onrender.com` | 本扩展服务端，发送用户主动输入的练习文本并取回 AI 结果。 |
| `host: translate.googleapis.com / mymemory` | 基础中英互译。 |

**数据用途声明**：不收集个人身份信息、不出售、不追踪；用户主动操作时发送的文本仅用于生成当次结果，服务端不留存。
